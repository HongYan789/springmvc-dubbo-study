## 实现springmvc+dubbo的调用

* springmvc-dubbo-consumer   springmvc+dubbo的消费者
* springmvc-dubbo-provider   springmvc+dubbo的生产者
* springmvc-dubbox-consumer   springmvc+dubbox的消费者
* springmvc-dubbox-provider   springmvc+dubbox的生产者

其中springmvc-dubbo-consumer采用的ali的dubbo中dubbo方式来实现开放接口
其中springmvc-dubbox-consumer采用的dangdang的dubbox中rest方式来实现开放接口

有技术问题请移步此处讨论
